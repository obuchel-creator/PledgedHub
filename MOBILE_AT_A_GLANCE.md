## 💡 Pro Tips

```
✅ Best Practices
├─ Always test on real device
├─ Start with mobile sizes
├─ Use 44x44px buttons
├─ Keep input font 16px+
├─ Monitor 4G performance
└─ Gather user feedback

❌ Avoid
├─ Buttons smaller than 44px
├─ Input font smaller than 16px
├─ Hover-only interactions
├─ Horizontal scrolling
├─ Complex layouts on mobile
└─ Skipping device testing

⚡ Quick Commands
├─ Chrome Mobile: Ctrl+Shift+M
├─ Get IP: ipconfig
├─ Test: http://YOUR_IP:5173
└─ Check size: View → Developer Tools
```
