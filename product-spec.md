# Product Specification – PledgeHub

## Purpose
A pledge management system for organizations, with AI automation, mobile money, reminders, OAuth, and analytics.

## Core Features
- Pledge creation, tracking, and reminders
- User authentication (JWT + OAuth)
- Mobile money payments (MTN, Airtel)
- Email/SMS notifications
- AI-powered messaging and analytics
- Accounting and financial reports

## User Roles
- Donor
- Staff
- Admin

## Compliance & Security
- Data privacy (GDPR-like)
- Secure payments
- Audit trail for financial transactions

## Acceptance Criteria
- All features must match this spec before release
