## рҹ”„ Data Flow

### Saving Credentials
```
User enters: MTN API Key
      в¬ҮпёҸ
Form validated (all fields present)
      в¬ҮпёҸ
Sent to: POST /api/payment-settings/mtn
      в¬ҮпёҸ
Service.savePaymentSettings('mtn', data)
      в¬ҮпёҸ
Encrypt with AES-256-CBC
      в¬ҮпёҸ
Generate random IV
      в¬ҮпёҸ
Store as: iv:encryptedData in database
      в¬ҮпёҸ
Return: Success message (no credentials echoed)
      в¬ҮпёҸ
User sees: вң… "MTN settings saved successfully"
```

### Retrieving Credentials
```
User clicks: Load Settings
      в¬ҮпёҸ
GET /api/payment-settings
      в¬ҮпёҸ
Service.getPaymentSettings()
      в¬ҮпёҸ
Fetch encrypted data from database
      в¬ҮпёҸ
Decrypt with AES-256-CBC
      в¬ҮпёҸ
Load into form fields
      в¬ҮпёҸ
User sees: Populated form with saved credentials
```

```

      в¬ҮпёҸ

      в¬ҮпёҸ

      в¬ҮпёҸ

      в¬ҮпёҸ

      в¬ҮпёҸ

      в¬ҮпёҸ
```
