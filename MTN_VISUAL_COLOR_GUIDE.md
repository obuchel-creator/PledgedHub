## Color Usage Rules

```
WHEN TO USE GOLD (#FFCC00):
✓ Primary action buttons
✓ Active/selected states
✓ Key highlights and emphasis
✓ Focus indicators (rings)
✓ Status badges (completed, active)
✓ Important alerts and warnings
✓ Brand accent elements
✓ Hover states on navigation

WHEN NOT TO USE GOLD:
✗ Body text (too bright)
✗ Large background areas (overwhelming)
✗ Disabled states (use lower opacity instead)
✗ Subtle elements (use supporting colors)
✗ Error messages (use red instead)
✗ Success messages (use green instead)

WHEN TO USE BLACK (#000000):
✓ Navigation bar background
✓ Main body text
✓ Headings and titles
✓ Form labels
✓ Text on gold backgrounds
✓ Icons and borders
✓ Navigation items

WHEN TO USE SUPPORTING COLORS:
✓ Light gray (#CCCCCC) - secondary navigation text
✓ Dark gray (#333333) - secondary text, hover states
✓ Light gold (#FFD700) - gradients, subtle accents
✓ Dark gold (#FFB800) - darker hover/active states
```
